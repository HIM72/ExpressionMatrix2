Getting started on MacOS

A native port of the ExpressionMatrix2 software to MacOS is not available.

The easiest way to use ExpressionMatrix2 software on MacOS is via Docker. A Docker image for the ExpressionMatrix2 software is available in GitHub repository iosonofabio/organic-box-factory. Detailed instruction on how to use it are available here.

Alternatively, you can use VirtualBox to create a virtual machine running Ubuntu 16, and use the ExpressionMatrix2 software inside the virtual machine.
