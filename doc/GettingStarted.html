Getting started

The ExpressionMatrix2 software runs "out of the box" and with minimal set up on the following platforms. For detailed information to get started using the ExpressionMatrix2 software, follow the link for the platform you are interested in.

  • Linux Ubuntu
  • Linux CentOS
  • Windows
  • MacOS

You can also run on other platforms, but that will require either building the code from source or using a virtualization tool such as VirtualBox or Docker. A Docker image for the ExpressionMatrix2 software is available in GitHub repository iosonofabio/organic-box-factory. Detailed instruction on how to use it are available here.

If you are planning to use Docker on Windows 10, keep in mind that on this platform Docker is mutually exclusive with VirtualBox: installing Docker will make VirtualBox unusable. For Windows 10, a much better choice is to use the Microsoft Ubuntu app - see here for more information.
