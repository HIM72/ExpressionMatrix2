Getting started

The ExpressionMatrix2 software runs "out of the box" and with minimal set up on the following platforms. For detailed information to get started using the ExpressionMatrix2 software, follow the link for platform you are interested in.

  • Linux Ubuntu
  • Linux CentOS
  • Windows

You can also run on other platforms, but that will require either building the code from source or using a virtualization tool such as VirtualBox or Docker.
